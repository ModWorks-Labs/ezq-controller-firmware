# BLOWER_MODE

Manual fan-only mode. Fan runs at 100% indefinitely until the user presses the button again. No GP involvement, no timeout.

## Enter

- Set LED to blower indicator (blue)
- Play mode-entry beep
- Command fan to 100%

## Update

- Monitor button: short press, transition to READY_IDLE

## Exit

- Command fan off
- Play mode-exit beep
